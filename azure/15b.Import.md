# 15. Import (Azure) <!-- omit in toc -->

# 1. Solicitar permisos en Azure
Confirma que tienes los permisos para crear Grupos de Recursos y Redes Virtuales (Virtual Network Contributor) para este laboratorio.

# 2. Crear la carpeta /import y prepararnos
Crea la carpeta de tu nuevo proyecto y establece tu configuración base editando el archivo `provider.tf`:
```tf
provider "azurerm" {
  features {}
}
```

# 3. (Instructor) Crear por CLI una nueva Virtual Network en Azure
El instructor crea recursos "por fuera" o a mano saltándose Terraform utilizando Azure CLI:
```vim
az group create --name miResourceGroupImport --location eastus

az network vnet create --resource-group miResourceGroupImport --name demo-vnet-import --address-prefixes 10.0.0.0/16 --subnet-name default
```
> Esta Red Virtual (VNet) **NO** es administrada actualmente por Terraform ni existe en tu archivo local. Por lo tanto debemos "importarla".

# 4. Buscar el ID oficial de la nueva VNet
Necesitas el identificador exacto de Azure ("Resource ID"). Ejecuta:
```vim
az network vnet show --resource-group miResourceGroupImport --name demo-vnet-import --query id -o tsv
```
> El ID de Azure lucirá similar a: `/subscriptions/xxxxxxxxx/resourceGroups/miResourceGroupImport/providers/Microsoft.Network/virtualNetworks/demo-vnet-import`

# 5. Crear un cascarón vacío para gobernar el recurso importado
Crea tu archivo `main.tf` y estipula la infraestructura. Por el momento déjalo en blanco.
```tf
resource "azurerm_virtual_network" "demo-import" {

}
```

# 6. Inicializar y validar el estado local (State)
```vim
terraform init
terraform show
```
> Retorna: **No state.** (Terraform dice "No conozco ninguna infraestructura todavía")

# 7. Importar la Instancia de Red utilizando su ID
Ejecuta el puente maestro entre el cascarón y el recurso flotante real, reemplazando el valor final con el ID extraído en el paso #4:
```vim
terraform import azurerm_virtual_network.demo-import <INGRESA_EL_ID_DEL_PASO_4>
```

Resultado esperado (éxito):
```vim
Import successful!

The resources that were imported are shown above. These resources are now in
your Terraform state and will henceforth be managed by Terraform.
```

# 8. Ejecutar el plan y observar la advertencia
```vim
terraform plan
```
Lanzará el error:
> **Error: Missing required argument**

> Explicación: El bloque `azurerm_virtual_network` está estructuralmente vacío. Terraform inyectó la realidad de la nube dentro de tu "State" pero **no autocompleta el código `main.tf`**. Tú debes rellenarlo manualmente.

# 9. Mostrar el Estado Actual
Solicita a Terraform que te muestre los atributos que absorbió de la vida real de Microsoft Azure:
```vim
terraform show
```

# 10. Completar el recurso `azurerm_virtual_network`
Utilizando la información mostrada por `terraform show` respecto a qué zona y nombres posee la red, completa tu código en `main.tf`:

```tf
resource "azurerm_virtual_network" "demo-import" {
  name                = "demo-vnet-import"
  location            = "eastus"
  resource_group_name = "miResourceGroupImport"
  address_space       = ["10.0.0.0/16"]
}
```

# 11. Plan & Check (Planear y contrastar)
Inspecciona si el terraform plan indica cambios destructivos:
```vim
terraform plan
```
> Importante: Debes velar que tu código cubra todos los flancos. Si faltara algún parámetro vital, el plan te advertirá de un "replace" (`- / +`), eliminando tu VNet real para reemplazarla por el molde del código local, destruyendo todo lo que hubiera colgado de ella.

# 12. Validar hasta que no haya desviaciones
Si el código encaja y es un reflejo fidedigno de la realidad que había en la nube, Terraform celebrará el empate de paridades:
> **No changes. Your infrastructure matches the configuration.**
>
> Terraform has compared your real infrastructure against your configuration and found no differences, so no changes are needed.

# 13. Eliminar el recurso desde Terraform
Como Terraform ha colonizado este recurso manual, ya eres el dueño técnico. Ejecuta su purga de rutina:
```vim
terraform destroy -auto-approve
```
> Destroy complete! Resources: 1 destroyed.
